Help

Windows systems

To install OOMapper on a Microsoft Windows system it is simply necessary to unzip the distribution file into a convenient directory. This directory should contain similar subdirectories and files as shown in the figure at the right.

A double click on the executable file Mapper.exe and the software will start with the Opening screen. A warning may appear advising that

The publisher could not be verified. Are you sure you want to run this software?


This appears because the compiled program has no digital signature from Microsoft. This free software is fully free.
If you are reasonably confident that the software has not been tampered with click to proceed.

Other systems

In addition to Windows, installation files for Linux (Ubuntu) may be found at http://oorienteering.sourceforge.net/ along with the source code which may be compiled to run on other systems.

To compile, have a look at the INSTALL file that comes with the source code.
