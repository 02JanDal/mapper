Help

Windows systems

To install OOMapper on a MS Windows system it is simply necessary to unzip the distribution file into a convenient directory. The directory Mapper-alpha1 should contain the subdirectories and files shown in the figure at right.

A double pick on the executable file Mapper.exe and the software will start with the Opening screen. A warning may appear advising that

The publisher could not be verified. Are you sure you want to run this software?
Blah! blah. blah ...various details

This appears because the compiled program has no digital signature from Microsoft. This free software is fully free.
If you are reasonably confident that the software has not been tampered with pick to proceed.

Other systems

In addition to Windows, installation files for Mac OS X operating system and Linux (Ubuntu) may be found at http://oorienteering.sourceforge.net/ along with the source code which may be compiled to run on other systems.

To compile it is necessary to have cmake and a recent Qt installed.
