Help

Settings

These controls will enable some features of the software to be set by you to suit your hardware and application.

General

The preferred language for the mapping interface and the help files may be selected here.

Editor

High quality map display

Your monitor may provide a better display if this check box is ticked. If your system is slow unchecking this may help.


Click tolerance

This tolerance is how close you need to be for a cut line point to be on the line. Set a larger value if you have difficulty making cuts, however some accuracy may be lost.


Auto select symbol

When this box is checked selection of any item on the map will cause the symbol editor to select the same symbol. This is often convenient as the symbol window may be left closed (showing more of the map on your screen) and symbols selected by simply clicking another of the same item elsewhere on your map.

Zoom to/from cursor

When ticked the zoom function will zoom toward or away from the cursor rather than from the middle of the window when using the scroll wheel. Using the menu or toolbar zoom tools (or the function keys F7 & F8) will still zoom to/from the middle of the screen.
