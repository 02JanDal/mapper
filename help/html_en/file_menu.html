Help

File menu

These controls provide standard file management tools. Keyboard shortcuts are shown thus: Ctrl + N will open a new map file.

New

Alt + N

This begins the creation of a new blank drawing, commencing with the dialog box to set the scale and the symbol set of a new map.

Open

Ctrl + O

Opens a directory to permit opening of an existing map file. This presently opens only files with the format .omap, or OCADv8. It is expected that other map formats may become available in the future.

Open Recent

This opens a list of recently opened files, recognising that mostly a file to be opened will usually have been previously opened in the recent past.

Save

Ctrl + S

Saves the currently opened file. If the file has not been previously saved a directory will open to enable the file to be saved with an appropriate name in a convenient location.

Save As

Saves the currently opened drawing as a new file - a directory will open to enable the file to be saved with an appropriate name in a convenient location.

Print

Ctrl + P

Prints the current map using available printers. Set the page orientation and size. The cooordinates of the area to be printed are shown on the print dialog in millimetres and the map screen shows a rectangle on the boundary of the print area. There is a button at the bottom of the dialog which displays a preview of the map as it will be printed

Close

Ctrl + W

Closes Open O-Mapper, after a file save if required.

Exit

Closes Open O-Mapper without saving the file.
