Scale all symbols

This dialog will re-scale all symbol editor dimensions. It does not rescale the length of lines or the boundaries of areas. It does rescale line widths and point symbol sizes. It is important to understand that use of this feature will create a departure from the ISOM standard symbols and the resulting map will thus not conform to the standard.
