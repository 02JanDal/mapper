Tools Menu

Editing is the process of marking the map with the features you wish to show, and adjusting the position and character of those features to provide a clear representation of the significant features on the ground.

The tools accessed through this menu are also available on the toolbar. The tool is the same whether it is invoked from the menu, from the toolbar or from the keyboard (press the letter key following each tool name to select the tool). Greater detail of the use of each tool may be found
